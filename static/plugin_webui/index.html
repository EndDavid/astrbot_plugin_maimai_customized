更新 B50 绑定机制

首次执行 更新b50 <SGWCMAID识别码> 成功后，会保存该用户的机台用户信息。之后可直接发送 更新b50 尝试复用旧绑定拉取成绩；如果失效，再重新发送新的 SGWCMAID。

锐评 B50

锐评会结合 Rating 分段、B35/B15、含金量、曲师/谱师与人格样本生成图片。专用模型只能在 AstrBot 插件配置页设置。

编辑人格

人格名称和聊天样本必填。系统会自动追加新样本、跳过重复或无效样本，并通过 LLM 智能筛选最优样本。

导入 JSON 聊天记录

只提取目标 QQ 的纯文本消息，并过滤 at、图片、引用/回复、表情、语音等内容。导入后系统会自动筛选最优样本。
